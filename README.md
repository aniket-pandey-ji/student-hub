
# 🎓 Student Hub

> An all-in-one academic management platform for BCA/B.Tech students — track attendance, manage projects, organize notes, and collaborate with peers.

---

## 🌐 Live Demo

[View Student Hub →](https://student-hub-jet.vercel.app)

## 🚀 Features

- 🏠 **Landing Page** — Hero section with CTA, feature grid, dashboard preview, testimonials
- 📊 **Dashboard** — Personalized view with upcoming deadlines, attendance %, recent scores
- 📅 **Attendance Tracker** — Semester-wise attendance monitoring with visual progress circle
- 🗂️ **Project Manager** — Team collaboration, task assignment, deadlines
- 📝 **Notes System** — Upload PDFs, store links, tag by subject
- 📚 **Subjects & Courses** — 14 BCA subjects with dedicated pages
- 🧠 **MindMap** — Visual learning tool
- 🏅 **Gamification** — Badges and points for consistency
- 🌙 **Dark / Light Mode** — Persisted via `localStorage` + system preference detection
- 📱 **Responsive** — Mobile-first design with hamburger sidebar
- ♿ **Accessible** — ARIA roles, labels, semantic HTML throughout
- 🔔 **Notification Badge** — Demo notification system
